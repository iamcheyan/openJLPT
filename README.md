# OpenJLPT

OpenJLPT 是一个开源、完全离线的日语能力测试（JLPT）题库项目。本项目致力于提供一个纯粹、高效且无干扰的学习环境，支持从 N1 到 N5 的全等级练习。

## 项目特点

- **完全离线运行**：采用纯静态架构，无需后端服务器，不依赖网络连接。所有逻辑均在本地浏览器或移动端 WebView 中闭环运行。
- **AI 生成题库**：项目中的所有题目、选项及解析均由大语言模型（AI）生成。这确保了题目的多样性，且不涉及任何现有教辅材料的版权纠纷。
- **多端支持**：
    - **网页端**：支持在任何现代浏览器中直接运行。
    - **安卓客户端**：提供基于原生 WebView 的安卓应用，支持全屏沉浸模式，适合专注学习。
- **针对性优化**：虽然支持各类设备，但界面设计特别针对**电子墨水屏 (E-ink)** 进行了优化（高对比度、无动画），在 Kindle、文石、Kindle 等设备上拥有极佳的显示效果。
- **智能振假名**：集成离线 `kuromoji.js` 引擎，实现汉字读音的实时标注。

## ⚠️ 重要声明

由于本项目题库内容全部由 **AI 生成**，尽管经过了一定程度的自动化筛选，但仍可能存在语言表达、逻辑瑕疵或语法错误。

**本项目仅供备考练习参考，不保证内容的 100% 准确性。**

## 快速开始

### 网页版
在根目录下运行本地服务器即可访问：
```bash
python3 server.py
```

### 安卓版
- 你可以从项目的 **GitHub Actions** 中下载自动编译好的最新版 APK。
- 题库数据默认存放于手机路径：`/sdcard/OpenJLPT/data`。

## 项目结构

- `data/`: 各等级 JSON 格式题库数据。
- `android/`: 安卓原生壳子源码。
- `assets/`: 核心引擎、CSS 样式、静态库及离线词典。
- `tools/`: 用于生成、同步与维护题库的脚本工具。
- `docs/`: 项目规划与技术文档。
- `index.html`: 练习生成引擎入口。
