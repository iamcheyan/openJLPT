# N2 题库数据文件说明

本目录包含 JLPT N2 级别的试题题库，由 `generate_bank.py` 自动生成，经审核模型审核后写入。
所有文件均为 JSON 格式，顶层结构包含 `meta`（元信息）和 `questions`（题目数组）。

---

## 文字・語彙（文字词汇）

| 文件 | 题型名称 | 说明 | 当前题数 |
|------|----------|------|----------|
| `vocab_reading.json` | 漢字の読み方 | 给出含汉字的词语，从四个选项中选择正确的读音。例：《謙虚》→ けんきょ | 89 |
| `vocab_kanji.json` | 漢字の表記 | 给出假名注音，从四个汉字写法中选择正确的表记。例：けいさい → 掲載 | 14 |
| `vocab_context.json` | 文脈規定 | 在句子的括号中填入最合适的词语，考查词汇在语境中的运用。 | 18 |
| `vocab_synonym.json` | 言い換え・類義 | 选择与划线词语意义最接近的近义词。例：深刻 → 深遠な | 16 |
| `vocab_usage.json` | 用法 | 给出一个目标词，从四个句子中选择该词使用最正确的一句。 | 16 |

## 通用字段说明

每道题目包含以下字段：

| 字段 | 说明 |
|------|------|
| `id` | 题目唯一标识，格式 `n2-{question_type}-{序号}` |
| `level` | 难度等级，固定为 `N2` |
| `question_type` | 题型代码，与文件名一致 |
| `sentence` / `passage` | 题干文本（单句题用 `sentence`，阅读题用 `passage`） |
| `target` | 目标词（仅词汇题，标记需要关注的词语） |
| `options` | 四个选项（数组） |
| `answer` | 正确答案的索引（0-3） |
| `explanation` | 答案解析 |
| `generated_by` | 生成该题的模型标识（如 `ark:Doubao-Seed-2.0-Pro`、`zhipu:GLM-4-Flash`、`mimo:mimo-v2.5-pro`） |
| `reviewed_by` | 审核模型标识 |
| `review_explanation` | 审核模型的评审意见 |
| `verified` | 是否通过审核（`true` / `false`） |
| `created_at` | 生成时间（ISO 8601） |

## 生成状态追踪

`data/generation-state/n2-vocabulary.json` 是词汇题的生成进度文件，记录每个词条在五种词汇题型中的生成状态：

- `pending` — 尚未生成
- 每种题型还记录 `*_attempts`（重试次数）和 `*_issue`（失败原因）

该文件用于断点续生成，确保程序中断后可从上次进度继续。
